Project

GroupUs GroupUs is a project I created to make it easier to put students into teams. It helps to make sure everyone gets a fair chance and that no one ends up in the same group all the time. GroupUs uses special computer tricks to randomly pick who goes where, which saves time for teachers and organizers.
